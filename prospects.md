## HOW TO USE THIS FILE

Each prospect has a **Web Status** rating:
- 🔴 **NO WEBSITE** — pure cold pitch, new build
- 🟠 **BROKEN** — website is down or unreachable, urgent pitch
- 🟡 **WEAK** — Wix / Weebly / GoDaddy / builder site, ripe for redesign
- 🟢 **OK** — proper website, lower priority unless you spot clear weaknesses

Priority is marked **⭐⭐⭐ HIGH**, **⭐⭐ MEDIUM**, or **⭐ LOW**.

## NOTES & IDEAS

- **Best cold call niches:** Barbers and plumbers respond well to "you came up in a local Facebook group" opener
- **Best email niches:** Painters, salons, takeaways — more likely to have an email on file
- **No website = new build pitch** (higher value, easier sell — they can't say "we like what we have")
- **Broken/Wix site = redesign pitch** (show them the problem first, then the solution)
- Run all prospects through the pipeline: research → build → deploy → email
- Check Facebook for businesses posting actively but with no linked website — these are warm targets
